# 📚 Exercícios06.JS

Atividade prática de JavaScript desenvolvida para a disciplina de **Sistemas Web** — cobrindo desde manipulação de strings até desafios com algoritmos.

---

## 🗂️ Conteúdo

| # | Tema | Exercícios |
|---|------|-----------|
| 1 | 🔤 Strings | Inverter string, contar vogais, palíndromo, capitalizar palavras |
| 2 | 📦 Objetos | Ficha do aluno, calculadora de IMC, lista de contatos |
| 3 | 🔀 Switch & Ternário | Dia da semana, calculadora, verificar aprovação |
| 4 | ⚡ Arrow Functions & Template Literals | Conversor de moeda, mensagem personalizada |
| 5 | 📋 Métodos de Array | map, filter, find |
| 6 | 🏆 Desafios | Validar senha, Cifra de César, ordenar alunos, remover duplicados, relatório da turma |

---

## 🚀 Como executar

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/Atividade06.git
```

2. Abra o arquivo `teste.html` no navegador

3. Pressione **F12** para abrir o Console

4. Chame qualquer função diretamente no Console:
```js
inverterString("javascript")     // "tpircsavaj"
calcularIMC("João", 75, 1.80)    // { nome: "João", imc: 23.15, ... }
cifrarCesar("hello world", 5)    // "mjqqt btwqi"
```

## 📝 Funções disponíveis

```js
// Strings
inverterString(texto)
contarVogais(texto)
ehPalindromo(texto)
capitalizarPalavras(frase)

// Objetos
criarAluno(nome, idade, curso)
calcularIMC(nome, peso, altura)
buscarContato(contatos, nome)

// Switch & Ternário
diaDaSemana(numero)
calculadora(num1, num2, operacao)
verificarAprovacao(nota)

// Arrow Functions
realParaDolar(reais, cotacao)
gerarMensagem(nome, idade, cidade)

// Métodos de Array
dobrarValores(numeros)
filtrarMaiores(pessoas)
buscarProduto(produtos, id)

// Desafios
validarSenha(senha)
cifrarCesar(texto, deslocamento)
ordenarPorNota(alunos)
removerDuplicados(array)
relatorioTurma(alunos)
```
